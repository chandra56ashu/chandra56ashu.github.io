**https://chandra56ashu.github.io**

## Run locally

Clone the repository and open `index.html` in a browser:

```bash
git clone https://github.com/chandra56ashu/chandra56ashu.github.io.git
cd chandra56ashu.github.io
```

For a local web server, run:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

The site is published automatically through GitHub Pages from the repository's default branch. The `.nojekyll` file ensures the static files are served directly.
